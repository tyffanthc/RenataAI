# GLOBAL_VALUE_AT_RISK.md

### Cel
Opisac jak Renata komunikuje Value at Risk (VaR) w sposob praktyczny.

### Definicja
VaR to szacowana waga potencjalnej straty przy danej decyzji.

### Zasady
- VaR zawsze jest prezentowany razem z kontekstem ryzyka.
- Komunikat ma byc zrozumialy dla pilota (bez zbednego zargonu).
- Niska pewnosc danych obniza zaufanie do VaR i jest sygnalizowana.

### Uzycie
- Priorytetyzacja decyzji: co tracisz, jesli to zignorujesz.
- Porownanie opcji w Decision Space (2-3 sensowne wybory).

### Jedno zdanie
> VaR pokazuje cene bledu, zebys mogl podjac swiadoma decyzje.
