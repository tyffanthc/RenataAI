# UX_INSIGHT_ALERTS.md


### Cel
Komunikaty interpretujące dane (biologia, jackpot, wartościowe cele).

### Definicja
Krótki komunikat będący **wnioskiem**, nie powtórzeniem HUD.

### Przykłady
- „Ta planeta jest warta skanowania.”
- „Możliwy jackpot biologiczny.”

### Zasady
- Jedno zdanie.
- Jeden raz na obiekt.
- Progi wartości.

### Jedno zdanie
> HUD pokazuje dane. Renata mówi, co z nich wynika.
