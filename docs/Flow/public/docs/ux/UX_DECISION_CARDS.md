# UX_DECISION_CARDS.md


### Cel
Standaryzacja krótkich, jednoetapowych decyzji użytkownika.

### Zasady
- Jedno pytanie.
- Max 3 opcje.
- Zawsze „Pomijam”.
- Jedna karta naraz.

### Kiedy używać
- Brak kluczowej informacji.
- Wybór priorytetu.
- Potwierdzenie ryzyka.

### Jedno zdanie
> Decision Cards umożliwia jedną decyzję i znika.
